Intro

This page is about how to use command blocks in Minecraft.

If you want to make mischief, have fun, and have knowledge that most people don't know, than keep reading!

What this is about

Command blocks are kind of how they sound, because you can input commands for all kinds of crazy stuff(and it's packed into a block)

The Basics

One of the first things you want to do is probably get a command block.

Unlike most blocks, you can't craft it or get it from your inventory, you have to get by a command.

The command to get one is:

/give @p command_block

Some basic commands to learn first are probably basic teleportation commands, e.g.

/teleport @p 200 70 100
